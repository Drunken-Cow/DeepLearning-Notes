### Books

这几年，关于人工智能、机器学习、深度学习的书如雨后春笋般，真的很多。下面列举一些大家普遍评价很高以及我有大概浏览过觉得不错的书，当个参考和记录：

- 黄文坚/唐源《TensorFlow实战》
